# VADY — تطبيق مراقبة أسعار العملات الرقمية

تطبيق أندرويد لمراقبة أسعار العملات الرقمية وتلقي تنبيهات سعرية، يعمل بالكامل محلياً على الجهاز بدون سيرفر خارجي (باستثناء طلبات الأسعار من CoinGecko).

## المزايا

- مراقبة أسعار مباشرة لأي عملة رقمية عبر CoinGecko API
- تنبيهات سعرية (فوق/تحت سعر مستهدف) — الحقل يُعبَّأ تلقائياً بالسعر الحالي
- **فحص أسعار حقيقي بالخلفية** كل 15 دقيقة، حتى لو كان التطبيق مغلقاً تماماً
- إشعارات نظام حقيقية (وليست مجرد Web Notifications تتوقف بإغلاق التطبيق)
- إشعار إضافي احتياطي عبر بوت تيليجرام خاص بالمستخدم
- رسم بياني (خط / شموع) لكل عملة مع مدد زمنية متعددة
- مؤشر الخوف والطمع (Fear & Greed Index)
- تخزين محلي بالكامل عبر SQLite على الجهاز — لا بيانات تُرسل لأي سيرفر خارج CoinGecko وتيليجرام (إن فُعّل)

## التقنية المستخدمة

المشروع ملف HTML واحد (`VADY_iappyxOS.html`) مصمم للعمل داخل منصة
[**iappyxOS**](https://github.com/iappyx/iappyxOS) — أداة تحوّل HTML/JS إلى تطبيق أندرويد حقيقي
(APK موقّع) مباشرة على الجهاز، بدون Android Studio أو Gradle أو أي بيئة بناء خارجية.

بدل الاعتماد على واجهات الويب القياسية (والتي تتوقف بمجرد إغلاق التطبيق)، يستخدم VADY الجسر
الأصلي لـ iappyxOS (`window.iappyx`) للوصول لقدرات النظام الحقيقية:

| الحاجة | الحل في VADY |
|---|---|
| تخزين البيانات | `iappyx.sqlite` (قاعدة بيانات SQLite حقيقية على الجهاز) |
| طلبات الشبكة | `iappyx.httpClient.request()` (وليس `fetch`، المحظور من أصل `file://`) |
| الإشعارات | `iappyx.notification.send()` (إشعار نظام حقيقي) |
| **فحص الأسعار بالخلفية** | `iappyx.tasks.schedule()` — مهمة مجدولة تعمل كل 15 دقيقة على الأقل، حتى والتطبيق مغلق |

### لماذا هذا مهم

أي تطبيق ويب (HTML/JS عادي أو WebView بسيط) يعتمد على `setInterval` أو `Notification` Web API
يتوقف عن العمل بمجرد إغلاق التطبيق أو انتقاله للخلفية لفترة طويلة — هذا قيد أساسي في المنصة
وليس خللاً في الكود. الحل الوحيد الحقيقي هو معالجة خلفية على مستوى نظام التشغيل، وهو ما توفره
`iappyx.tasks` هنا.

## طريقة التشغيل

1. ثبّت [iappyxOS](https://github.com/iappyx/iappyxOS/raw/main/bin/iappyxOS.apk) على جهاز أندرويد (Android 10+)
2. افتح iappyxOS ← Create ← Paste HTML
3. الصق محتوى `VADY_iappyxOS.html`
4. اضغط Build — سيُنشأ تطبيق VADY كتطبيق أندرويد حقيقي بأيقونة خاصة في قائمة التطبيقات

## قيود معروفة

- الحد الأدنى لفترة فحص الأسعار بالخلفية هو **15 دقيقة** (قيد يفرضه نظام أندرويد لتوفير البطارية عبر `iappyx.tasks`، وليس قابلاً للتخطي)
- مدة تنفيذ المهمة الخلفية محدودة بـ 30 ثانية — كافية لفحص قائمة مراقبة معتدلة الحجم
- مصدر الأسعار هو CoinGecko العام (بدون مفتاح API)، وقد يطبّق حدود معدل طلبات (Rate Limiting) في حالات الاستخدام المكثف
